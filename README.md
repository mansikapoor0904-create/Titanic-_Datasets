# Titanic-_Datasets
Task 2:  Exploratory Data Analysis (EDA)
📊 Titanic Dataset – Exploratory Data Analysis (EDA)
📌 Project Overview

This project is part of an AI & ML Internship Task focused on Exploratory Data Analysis (EDA).
The goal is to understand the dataset using statistics and visualizations to uncover patterns, trends, and insights.

🎯 Objective
Analyze dataset using descriptive statistics
Visualize data distributions
Identify patterns and relationships
Detect anomalies and outliers
Generate meaningful insights
🛠 Tools & Libraries Used
Python
Pandas
Matplotlib
Seaborn
Plotly
📂 Dataset
Dataset used: Titanic Dataset
Source: Kaggle
🔍 Steps Performed
1. Data Understanding
Loaded dataset using Pandas
Viewed structure using .head(), .info(), .describe()
2. Summary Statistics
Calculated:
Mean
Median
Standard Deviation
Min & Max values
3. Data Visualization
📈 Univariate Analysis
Histograms to understand distribution
Boxplots to detect outliers
📊 Bivariate Analysis
Pairplot to study relationships between features
Correlation matrix using heatmap
4. Pattern & Trend Analysis
Observed survival trends based on:
Gender
Age
Passenger class
5. Outlier & Skewness Detection
Identified skewed features
Detected extreme values using boxplots
📊 Visualizations Included
Histograms
Boxplots
Pairplots
Heatmap (Correlation Matrix)
Count plots
📈 Key Insights
Female passengers had higher survival rates
Higher-class passengers were more likely to survive
Age distribution shows most passengers were young adults
Some features showed skewness and outliers
